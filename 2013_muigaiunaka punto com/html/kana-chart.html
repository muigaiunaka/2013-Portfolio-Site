Kana Chart

Typography II

Fall 2014

For the class, Typography 2, each student was tasked with designing an A1 sized (23.4 inch x 33.1 inch) educational poster that would be used to teach American, middle school-aged children the basics of a non-Roman alphabetic writing system. The phonetic Japanese script of Kana was chosen because of Muigai's connection with the Japanese language (studied Japanese for a year). The poster consists of two vertical charts showing the Hirigana characters and the Katakana characters. Below each kana character is the Roman script equivalents. To the left of each column of characters is a brief description of the writing script and a fun fact about it's origins. Underneath all the writing system informational is a representational, minimalist geisha icon that Muigai designed using the pen tool in Adobe InDesign. The spacing of each column of script was created using vertical tabs. The typefaces used are Gill Sans MT and Noto Sans Japanese.
